# SQL_data_warehouse_project
A data warehouse built with SQL Server, including ETL processes, data modeling, and analytics techniques. This project is designed as a portfolio project and follows industry best practices for data analytics.

#  Project overview
The project involves: 
#### 1. Data architecture: 
Desingning a modern data warehouse using Medalion architecture, Bronze, Silver and Gold layer
#### 2. ETL Pipelines: 
Extracting, transforming and loading data from source system into the warehouse
#### 3. Data modeling: 
Developing fact and dimension tables optimized for analytical queries.
#### 4. Analytics: 
Creating SQL reports and dashboards for actionable insights.

 # Specifications
 - Data sources: Import data from two sources systems (ERP and CRM ) as CSV files(Files on docs         section)       
 - Data quality: Cleanse and handle data quality issues before loaded on the final layer (gold          layer)
 - Integration: Combine both sources using normalization and standardization to ensure user‑friendly reporting and analytics.
 - Documentation: Provide clear documentation on the roadmap followed to create the data warehouse

#### Bronze layer: 
  Stores raw data just like it comes from the source system, the goal is convert this data from a    CSV file into a SQL that we can query anb work with it
#### Silver layer:
  This layer handles data cleaning and transformation (normalization, standardization, handling invalid data, nulls, etc.).
#### Gold layer:
  Data is ready to analytics and reporting based on the business needs. Data is modeled into a star schema with dimention and fact views.
